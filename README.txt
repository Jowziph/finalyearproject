Multigrid preconditioned conjugate gradient method for solving the 2D sign positive Helmholtz equation.
Edit parameters.dat to experiment with other set-ups.
precision == 1: multigrid uses single precision (32 bit) computations.
precision == 2: multigrid uses double precision (64 bit) computations.

Type "make" then "./test5.x" or "./test6.x" to see program run.
